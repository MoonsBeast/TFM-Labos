# TFM-Labos

## Instalation

1. Install Python 3, Docker and Docker-compose

2. Clone the repository:

```
git clone https://github.com/MoonsBeast/TFM-Labos
```

3. Access folder:

```
cd TFM-Labos
```

4. Create a virtual env (optional):

```
python3 -m venv venv
source venv/bin/activate
```

5. Install requirements

```
pip install .
```
